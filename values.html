CORE VALUES

가치

어른이아가 절대 타협하지 않는 네 가지.

어른이아 Core Values는 단순한 슬로건이 아닙니다.
우리가 콘텐츠를 만들고, 커뮤니티를 운영하고, 참여자를 대하는 방식 전체를 결정하는 기준입니다.

01

연습할 권리

어른도 연습할 수 있어야 합니다. 서툰 건 결함이 아니라 과정입니다.

  • "이미 다 알겠지"라고 가정하지 않는다
  • 서툼은 결함이 아니라 과정이다
  • 준비 없이 던져지는 걸 당연하게 여기지 않는다
  • 어른에게도 연습할 권리가 있다
02

자유와 기준의 공존

자유는 방임이 아닙니다. 선택은 자유롭되, 기준은 숨기지 않습니다.

  • 선택은 자유롭다
  • 기준은 숨기지 않는다
  • 책임은 경험 안에서 배운다
  • 마음대로가 아니라, 의미 있게
03

결정의 주체성

우리는 대신 결정하지 않습니다. 스스로 판단하도록 돕습니다.

  • 추천하지 않는다
  • 강요하지 않는다
  • 스스로 판단하도록 돕는다
  • 선택의 결과까지 내 것이 되도록
04

존중

참여자를 '미완성 어른'으로 보지 않습니다. 어른을 어른으로 대합니다.

  • 가르치려 들지 않는다
  • 내려다보지 않는다
  • 동등한 위치에서 함께 설계한다
  • 어른을 어른으로 대한다
IN PRACTICE

가치는 이렇게 적용됩니다

콘텐츠 제작

전문가에게 "성공 비결"이 아닌, "처음엔 어땠냐"를 묻습니다. 정답이 아닌 과정을 담습니다.

커뮤니티 운영

라운드테이블에서 진행자는 방향을 강요하지 않습니다. 서로의 이야기를 듣고 스스로 발견합니다.

멤버 관계

어른이아는 "이렇게 해야 해"라고 말하지 않습니다. "이런 선택지가 있어"라고 보여줄 뿐입니다.
